# 课题名称

LLM Agent 深度搜索框架设计与实战优化

# 课题难度

（1最易-5 最难）4

# 课题描述（即研究内容）：

本课题要求基于给定 Simple Agent 框架，设计并实现一个面向 Deep
Research 任务的 Agent Harness，使基础 Agent 具备长程检索、证据整合、可复现评测和过程诊断能力。核心研究内容包括：
1. Deep Research 流程设计：设计合理的 Agent Loop，实现证据去重、来源可信度判断、引用追踪、冲突信息处理与最终答案生成，以及 Agent 内部对错误信息的循环跟踪、方案迭代；
2. Agent Harness 总体架构设计：在 Simple Agent 之外实现问题解析、分解，工具调用，失败重试与状态记录等多种任务能力；
3. 长程任务执行与过程可观测性：保存每一步查询、页面、证据片段、工具调用、模型判断与中间结论，支持断点恢复、错误归因；
4. 端到端效果测试：在 Long-Horizon 数据集上进行端到端效果测试，重点考察答案准确率、证据覆盖率、引用可验证性、工具调用效率、成本与时延。

# 汇报时需呈现的关键要点 & 创新点：

1. 长程研究策略的实现细节：任务分解、动态查询生成、多跳证据收集、来源可靠性判断、冲突证据处理和引用生成方式；
2. Deep Research Agent 系统架构设计规范、模块间接口设计、内部处理循环逻辑，特别是对于错误步骤的归因分析与自主迭代；
3. 多种 Long-Horizon 任务上的定量测试结果（任务完成率、Token 使用量、运行时间、失败类型分布等）。

# 先修知识及技能要求：

1. 掌握 LLM 基础原理、Agent 框架（如 LangChain、AutoGPT）核心逻辑和工具调用机制；
2. 熟悉信息检索、网页解析、查询改写、多跳问答等基础实现方法；
3. 具备 Python 编程能力，能够基于给定框架完成二次开发与测试
4. 了解 Agent 评估方法，能够设计 baseline、消融实验和错误析；
5. 具备基本工程意识，包括配置管理、结果缓存、API 成本控制和实验可复现。
